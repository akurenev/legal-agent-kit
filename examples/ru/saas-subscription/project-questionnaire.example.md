

# Пример заполненной анкеты проекта: SaaS с подпиской

## Назначение примера

Этот пример показывает, как AI-агент может заполнить анкету проекта для типового SaaS-сервиса с регистрацией, личным кабинетом, тарифами, оплатой, подпиской, пробным периодом, уведомлениями, API и внешними интеграциями.

Пример не является готовым юридическим документом. Все данные ниже демонстрационные.

### 6.2. Цели обработки данных

- регистрация и авторизация;
- предоставление доступа к сервису;
- управление аккаунтом организации;
- обработка заявок и событий;
- отправка сервисных уведомлений;
- прием оплаты и выдача чеков;
- поддержка пользователей;
- работа API и webhook;
- подключение внешних интеграций;
- аналитика использования сервиса;
- безопасность и диагностика ошибок.

| Тип данных | Срок хранения | Основание / комментарий |

| API-токены | До удаления токена или отключения интеграции | Токены должны быть защищены |
